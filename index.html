Contexto del Proyecto

Desarrollo de un juego clásico utilizando programación orientada a objetos

Objetivo

Desarrollar un juego del ahorcado interactivo que demuestre los principios de la Programación Orientada a Objetos, implementando encapsulación, herencia y polimorfismo en Python.

Justificación

El juego del ahorcado es ideal para aplicar conceptos de POO, permitiendo la creación de clases bien definidas, manejo de estados y encapsulación de la lógica del juego.

Tecnología

Implementación en Python utilizando programación orientada a objetos, con interfaz de consola y manejo de archivos para el diccionario de palabras.

JuegoAhorcado

Descripción

Clase principal que controla la lógica del juego. Maneja el estado de la partida, las letras adivinadas, los intentos restantes y coordina las interacciones entre las demás clases.

Atributos principales:
  • palabra_secreta: Palabra a adivinar
  • letras_adivinadas: Lista de letras ya intentadas
  • intentos_restantes: Contador de intentos fallidos
  • estado_juego: Estado actual (jugando, ganado, perdido)
JuegoAhorcado.py 
class JuegoAhorcado:
    def __init__(self):
        self.palabra_secreta = ""
        self.letras_adivinadas = []
        self.letras_incorrectas = []
        self.intentos_restantes = 6
        self.estado_juego = "jugando"
        self.diccionario = DiccionarioPalabras()
        self.interfaz = InterfazConsola()
    
    def iniciar_juego(self):
        """Inicia una nueva partida del juego"""
        self.palabra_secreta = self.diccionario.obtener_palabra_aleatoria()
        self.letras_adivinadas = []
        self.letras_incorrectas = []
        self.intentos_restantes = 6
        self.estado_juego = "jugando"
        self.interfaz.mostrar_mensaje_bienvenida()
    
    def procesar_letra(self, letra):
        """Procesa la letra ingresada por el usuario"""
        if letra in self.letras_adivinadas or letra in self.letras_incorrectas:
            return "Ya has usado esta letra"
        
        if letra in self.palabra_secreta:
            self.letras_adivinadas.append(letra)
            if self._palabra_completa():
                self.estado_juego = "ganado"
        else:
            self.letras_incorrectas.append(letra)
            self.intentos_restantes -= 1
            if self.intentos_restantes == 0:
                self.estado_juego = "perdido"
    
    def _palabra_completa(self):
        """Verifica si la palabra ha sido completamente adivinada"""
        return all(letra in self.letras_adivinadas for letra in self.palabra_secreta)
    
    def obtener_palabra_mostrar(self):
        """Retorna la palabra con las letras adivinadas visibles"""
        return ''.join(letra if letra in self.letras_adivinadas else '_' 
                      for letra in self.palabra_secreta)

DiccionarioPalabras

Descripción

Clase encargada de manejar el diccionario de palabras del juego. Se encarga de cargar las palabras desde un archivo, seleccionar palabras aleatorias y gestionar diferentes categorías de dificultad.

Atributos principales:
  • palabras: Lista de palabras disponibles
  • archivo_diccionario: Ruta del archivo de palabras
  • categoria_actual: Categoría de dificultad seleccionada
DiccionarioPalabras.py 
import random
import os

class DiccionarioPalabras:
    def __init__(self, archivo_diccionario="palabras.txt"):
        self.archivo_diccionario = archivo_diccionario
        self.palabras = []
        self.categorias = {
            "facil": [],
            "medio": [],
            "dificil": []
        }
        self.cargar_palabras()
    
    def cargar_palabras(self):
        """Carga las palabras desde el archivo de diccionario"""
        try:
            if os.path.exists(self.archivo_diccionario):
                with open(self.archivo_diccionario, 'r', encoding='utf-8') as archivo:
                    self.palabras = [palabra.strip().upper() 
                                   for palabra in archivo.readlines() 
                                   if palabra.strip()]
                self._categorizar_palabras()
            else:
                self._crear_diccionario_por_defecto()
        except Exception as e:
            print(f"Error al cargar el diccionario: {e}")
            self._crear_diccionario_por_defecto()
    
    def _categorizar_palabras(self):
        """Categoriza las palabras por dificultad según su longitud"""
        for palabra in self.palabras:
            if len(palabra) <= 4:
                self.categorias["facil"].append(palabra)
            elif len(palabra) <= 7:
                self.categorias["medio"].append(palabra)
            else:
                self.categorias["dificil"].append(palabra)
    
    def obtener_palabra_aleatoria(self, categoria="todas"):
        """Obtiene una palabra aleatoria del diccionario"""
        if categoria == "todas":
            return random.choice(self.palabras) if self.palabras else "PYTHON"
        else:
            palabras_categoria = self.categorias.get(categoria, self.palabras)
            return random.choice(palabras_categoria) if palabras_categoria else "PYTHON"
    
    def _crear_diccionario_por_defecto(self):
        """Crea un diccionario básico si no existe el archivo"""
        palabras_por_defecto = [
            "PYTHON", "PROGRAMACION", "COMPUTADORA", "ALGORITMO",
            "VARIABLE", "FUNCION", "CLASE", "OBJETO", "METODO",
            "HERENCIA", "POLIMORFISMO", "ENCAPSULACION"
        ]
        self.palabras = palabras_por_defecto
        self._categorizar_palabras()

InterfazConsola

Descripción

Clase responsable de manejar toda la interacción con el usuario a través de la consola. Se encarga de mostrar el estado del juego, dibujar el ahorcado, capturar las entradas del usuario y mostrar mensajes informativos.

Atributos principales:
  • dibujos_ahorcado: Lista con los dibujos del ahorcado
  • colores: Códigos de colores para la consola
  • simbolos: Símbolos especiales para el display
InterfazConsola.py 
import os
import sys

class InterfazConsola:
    def __init__(self):
        self.dibujos_ahorcado = [
            """
            ___________
            |         |
            |         
            |         
            |         
            |         
            |
            |_________
            """,
            """
            ___________
            |         |
            |         O
            |         
            |         
            |         
            |
            |_________
            """,
            """
            ___________
            |         |
            |         O
            |         |
            |         
            |         
            |
            |_________
            """,
            # ... más dibujos
        ]
        
        self.colores = {
            'rojo': '\033[91m',
            'verde': '\033[92m',
            'amarillo': '\033[93m',
            'azul': '\033[94m',
            'reset': '\033[0m'
        }
    
    def limpiar_pantalla(self):
        """Limpia la pantalla de la consola"""
        os.system('cls' if os.name == 'nt' else 'clear')
    
    def mostrar_mensaje_bienvenida(self):
        """Muestra el mensaje de bienvenida del juego"""
        self.limpiar_pantalla()
        print(f"{self.colores['azul']}")
        print("=" * 50)
        print("    🎮 BIENVENIDO AL JUEGO DEL AHORCADO 🎮")
        print("=" * 50)
        print(f"{self.colores['reset']}")
        print()
    
    def mostrar_estado_juego(self, juego):
        """Muestra el estado actual del juego"""
        print(f"{self.colores['amarillo']}Estado del Juego:{self.colores['reset']}")
        print("-" * 30)
        
        # Mostrar el dibujo del ahorcado
        intentos_fallidos = 6 - juego.intentos_restantes
        if intentos_fallidos < len(self.dibujos_ahorcado):
            print(self.dibujos_ahorcado[intentos_fallidos])
        
        # Mostrar palabra con guiones
        palabra_mostrar = juego.obtener_palabra_mostrar()
        print(f"Palabra: {' '.join(palabra_mostrar)}")
        print()
        
        # Mostrar letras usadas
        if juego.letras_incorrectas:
            print(f"{self.colores['rojo']}Letras incorrectas: {', '.join(juego.letras_incorrectas)}{self.colores['reset']}")
        
        if juego.letras_adivinadas:
            print(f"{self.colores['verde']}Letras correctas: {', '.join(juego.letras_adivinadas)}{self.colores['reset']}")
        
        print(f"Intentos restantes: {juego.intentos_restantes}")
        print("-" * 30)
    
    def solicitar_letra(self):
        """Solicita una letra al usuario"""
        while True:
            letra = input("Ingresa una letra: ").strip().upper()
            if len(letra) == 1 and letra.isalpha():
                return letra
            else:
                print("Por favor, ingresa solo una letra válida.")
    
    def mostrar_resultado_final(self, juego):
        """Muestra el resultado final del juego"""
        if juego.estado_juego == "ganado":
            print(f"{self.colores['verde']}")
            print("🎉 ¡FELICITACIONES! 🎉")
            print("¡Has ganado el juego!")
            print(f"La palabra era: {juego.palabra_secreta}")
            print(f"{self.colores['reset']}")
        else:
            print(f"{self.colores['rojo']}")
            print("💀 ¡GAME OVER! 💀")
            print("¡Has perdido el juego!")
            print(f"La palabra era: {juego.palabra_secreta}")
            print(f"{self.colores['reset']}")
    
    def preguntar_nuevo_juego(self):
        """Pregunta si el usuario quiere jugar de nuevo"""
        respuesta = input("¿Quieres jugar de nuevo? (s/n): ").strip().lower()
        return respuesta in ['s', 'si', 'sí', 'y', 'yes']
